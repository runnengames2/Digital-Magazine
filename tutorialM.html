Onderwerp: Variabelen

Leer stap voor stap hoe je variabelen kunt gebruiken in programmeertalen.

Wat moet er gebeuren

  • Het onderwerp moet bepaald worden
  • Er moet een prompt voor de AI gemaakt worden
  • Het eindresultaat moet minimaal 600 woorden zijn
  • De tutorial moet stap voor stap worden uitgelegd

De tutorial bevat

  • Wat het eindresultaat is
  • Welke stappen er nodig zijn om daar te komen
  • Wat de code doet
  • Waarom die code nodig is

Dit moet op de TutorialPagina

  • De AI prompt
  • De door AI gegenereerde tutorialtekst
  • Eventuele code voorbeelden
  • Korte reflectie wat je zelf hebt geleerd
